# Wokkibot-Go
Wokkibot rewritten yet again. This time in GO using Discordgo.

# Commands
##### Music related commands
* /play
* /skip
* /seek
* /volume
* /queue
* /pause (Disabled)
* /resume (Disabled)
##### Other commands
* /friday
* /roll

# Setup
* Get [Lavalink](https://github.com/freyacodes/Lavalink)
* Setup .env
* Run `go run main.go` or build
